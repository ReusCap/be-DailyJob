# be-DailyJob 

### 1. 오늘 완료한 것
- 오늘 실제로 끝낸 기능/수정/학습:
    - 직무 과제 코드는 아직 작성을 시작하지 못하였으나, 과제 구현 위한 서버 구조 학습하였다.
    - ServerSocket, Socket, InputStream, OutputStream역할과 클라이언트 요청 받고 응답 흐름 정리하였다.
    - Thread, Runnable, start(), run()차이 흐름 정리하였다.
    - ExecutorService와 스레드풀 개념 학습하였다.
    - Logger, try-with-resources, IOException 처리 방식 정리하였다. 
- (오늘의 목표 중에서) 아직 끝내지 못한 부분:
    - ExecutorService구조를 아직 과제 코드에 적용하지 못했다.
    - HTTP 요청 처리 부분 심화 학습이 필요하다.
